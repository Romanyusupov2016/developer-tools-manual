console.log({ obj1, obj2 }) - вывод объектов с указнанием их имён.
console.table({ obj1, obj2 }) - вывод содержимого объектов в виде таблицы.

console.warn() - вывод предупреждений.
console.error() - вывод ошибок.
console.info() - вывод информационных сообщений.

console.trace() - вывод результата трассировки стека.
console.count('метка') - вывод количества вызовов данного метода с заданной меткой.

console.time('метка') - таймер замеряющий промежуток времени проходящего от console.time() до console.timeEnd().
console.timeEnd('метка')

console.dir() - вывод DOM node.

$('selector') - возвращает первый DOM елемент.
$$('selector') - возвращает массив всех DOM елементов.
$_ - возвращает результат последнего выражения.

clear() - очистить консоль.
