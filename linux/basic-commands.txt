cd - переход в указанный каталог.
pwd - полный путь к текущему каталогу.
ls -R - рекурсивный вывод вложенных каталогов.
ls -al - показать скрытые файлы.
ls -lt - листинг по дате последнего изменения файлов.
ls -ltr - листинг по дате последнего изменения файлов, в обратном порядке.
ls -lhS - листинг по размеру файла.
ls -lhSr - листинг по размеру файла, в обратном порядке.
tree - показать дерево каталогов и файлов.
cat - вывести содержимое текстового файла.
less - вывести содержимое большого текстового файла, с постраничной прокруткой.
clear - очистка экрана.
exit - закрыть терминал.
man command - справка по комманде.
ctrl + a - перейти в начало строки.
ctrl + e - перейти в конец строки.
history n - показать историю комманд, с n-ым количеством последних записей.

touch - создание текстового файла.
mkdir - создание каталога.
mkdir -p - создать дерево каталогов.
rmdir - удаление пустого каталога.
rm - удаление файла.
rm -rf - рекурсивное удаление каталога и подкаталогов.
mv - переименование/перемещение файла или каталога.
cp - копирование файла.
ln -s обсолютный путь к файлу имя_ссылки - создание символической ссылки(Symbolic link).
ln file hard-link - создание жёсткой сылки(Hard link).

kill -9 PID -убить процесс.
command & - запуск задачи в фоновом режиме.
jobs - список запущенных фоновых задач.
fg n - перевести фоновую задачу с номером n в foreground(обычный режим).
sudo reboot - перезагрузить компьютер.
sudo halt - выключить компьютер.

stdin 0 - стандартный поток ввода.
stdout 1 - стандартный поток вывода.
stderr 2 - стандартный поток ошибок.
command > file - перенаправление вывода комманды в файл.
command >> file - перенаправление вывода комманды в файл, с добавлением данных в конец файла.
command 2> file - перенаправление вывода потока ошибок комманды в файл.
command 2>> file - перенаправление вывода потока ошибок комманды в файл (с добавлением данных).
command < file - перенаправить комманде ввод данных из файла.
command-1 | command-2 - перенаправить вывод комманды 1 на ввод комманды 2.

chown owner:group file - назначить владельца и группу владельца для файла.
chown -R owner:group directory - назначить владельца и группу владельца для каталога.
chmod 777 file - назначить права на файл.
chmod -R 777 directory - назначить права на каталог.
groups - список групп в которых состоит пользователь.
mode: owner|group|others
permissions:
Read = 4
Write = 2
Execute = 1
None = 0

passwd - изменить пароль пользователя. 

find - показать все файлы в текущей директории.
find path - показать все файлы в указзаной директории.
find path -name "grep-шаблон" - поиск файла по имени.
find path -iname "grep-шаблон" - регистро независимый поиск файла по имени.
find path -maxdepth 1 -name "grep-шаблон" - поиск файла только в указанной папке(в текущем уровне вложенности).
find path -maxdepth 1 -name ".*" - поиск скрытых файлов и папок.
find path -type f -empty - поиск пустых файлов.
find path -type d -empty - поиск пустых каталогов
grep шаблон file - поиск вхождения подстроки в файле по указанному шаблону.
grep -i шаблон file - регистро независимый поиск вхождения подстроки в файле по указанному шаблону.
whereis - поиск пути исполняемого файла, исходников, man-страниц программы.
which - поиск сиполняемого файла программы.

sudo updatedb - обновить базу данных, для использования команды locate.
locate шаблон - поиск файлов, используя проиндексированную базу данных.

sudo mount /dev/sdn1 /mnt/usb - монтировать устройство в каталок /mnt/usb.
sudo mount -t vfat /dev/sdn1 /mnt/usb - монтировать устройство с указанием файловой системы FAT32.
sudo mount -o users,uid=1000,gid=1000 /dev/sdn1 /mnt/usb - монтировать устройство и назначить владельцем для устройства - пользователя.
sudo umount /dev/sdn1 - размонтировать устройство.
