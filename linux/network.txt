ip addr show - информация о сетевых интерфейсах.
ip route - таблица маршрутизации.
curl ifconfig.me - узнать внешний ip адресс шлюза (точки доступа выхода в интернет).
sudo ss - показать все сетевые подключения.
sudo ss -t - показать tcp-соеднинения.

curl:
curl URL - выплнить GET запрос.
curl -i URL - получить заголовки HTTP ответа.
curl -I URL - получить только заголовки HTTP ответа, без тела ответа.
curl -d "param1=value1&param2=value2" -X POST URL - выполнить POST запрос.
curl -d '{"param1": "value1", "param2": "value2"}' -H "Content-Type: application/json" -X POST URL - выполнить POST запрос с передачей JSON данных.
curl -d "@file.json" -X POST URL - выполнить POST запрос с передачей JSON файла.
curl -L URL - выполнить GET запрос, с автмоматическим редиректом.
curl -o file.html URL - скачать файл и сохранить c указанным именем.
curl -O URL - скачать и сохранить файл.
curl --verbose -I URL - выполнить GET запрос, с детальным выводом информации о запросе и ответе.

wget:
wget -V | --version - версия программы.
wget -h | --help - справочная информация о программе.
wget URL - скачивание файла.
wget -O new_name URL - скачивание и сохранение файла под указанном именем.
wget -c URL - продолжить загрузку, при остановках загрузки.
wget -i file - скачать фйлы по URL, которые записанны в файле.
wget -p URL/index.html - скачивание сайта.

ssh:
ssh server runs on port 22.
ssh login@host - подключение к серверу SSH.
shh login@host -p port_number - подключение к серверу SHH по указанному порту.
ssh-keygen - генерация пары ключей:
    /home/user/.ssh/id_rsa - приватный ключ.
    /home/user/.ssh/id_rsa.pub - публичный ключ.
/home/user/.ssh/authorized_keys - файл хранящий список авторизованных публичных ключей (находится на сервере).
ssh-copy-id login@host - добавление содержимого публичного ключа в файл "authorized_keys" на сервере.
