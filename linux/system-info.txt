screenfetch - информация о системе.
arch - показать архитектуру процессора.
htop - улучшенная версия top.
top - просмотр запущенных процессов в интерактивном режиме.
ps -aux - список всех запущенных процессов.
uptime - время работы системы, после загрузки.
lscpu - информация о процессоре.
free -h - информация по ОЗУ и swap
df -h - информация об использовании дискового пространства файловой системы.
sudo du -sh directory - вывести размер, занимаемой директорией на диске.
uname -a - информация о ядре Linux.
lsb_release -a - информация о дистрибутиве Linux.
sudo fdisk -l /dev/sda - информация о разделах на диске.
sudo fdisk -l -информация о разделах.
lspci | grep -i vga - информация об видеокарте.
lsblk - список всех блочных устройств подключённых к системе.
date - отобразить текущую дату и время.
file - определяет тип файла.
env - переменные окружения системы.

Типы файлов:
- - обычные файлы.
b - файлы блочных устройств.
c - файлы символьных устройств.
l - символическая ссылка.
p - туннели (pipe).
s - сокеты.
d - каталоги.

/bin - стандартные утилиты.
/sbin - служебные утилиты.
/etc - конфигурационные файлы.
/etc/hosts - файл содержащий локальную базу DNS.
/dev - файлы устройств.
/proc - информация о процессах.
/var - часто изменяемые файлы.
/var/log - файлы логов.
/tmp - временные файлы.
/usr - программы пользователя.
/usr/bin - исполняемые файлы программ.
/usr/sbin - служебные программы.
/usr/lib - библиотеки программ.
/usr/local - файлы пользователя.
/boot - файлы загрузчика системы.
/lib - системные библиотеки.
/opt - проприетарные программы, драйвера.
/mnt - каталог для монтирования внешних, дополнительных файловых систем.
/media - каталог для монтирования внешних устройств (usb-флешки и т.д.).
/sys - файловая система организуемая ядром, содержит информацию о системе.

/home/user/.bashrc - конфигурационный файл переменных окужения для конкретного пользователя.
/etc/environment - конфигурационный файл переменных окружения на системном уровне.

Разметка диска с UEFI: 
1. sda1 fat32 EFI   512mb
2. sda2 ext4  /     20gb
3. sda3 ext4  /home ~

efibootmgr - показать настройки UEFI.
sudo efibootmgr -o 0,1,2,3 - изменить порядок загрузчиков.
